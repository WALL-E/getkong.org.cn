Installation:

  1. Enable the EPEL repository:

    Before installing Kong, you need to install the epel-release package for right version of your operating system, so that Kong can fetch all the required dependencies:

    $ EL_VERSION=`cat /etc/redhat-release | grep -oE '[0-9]+\.[0-9]+'` && \
  sudo yum install https://dl.fedoraproject.org/pub/epel/epel-release-latest-${EL_VERSION%.*}.noarch.rpm

  2. Install the Package:

    If you are downloading the package, execute:

    $ sudo yum install kong-0.9.2.*.noarch.rpm --nogpgcheck

  3. Configure your database

    Configure Kong so it can connect to your database. Kong supports both PostgreSQL 9.4+ and Cassandra 2.2.x as its datastore.

    If you are using Postgres, please provision a database and a user before starting Kong, ie:

    CREATE USER kong; CREATE DATABASE kong OWNER kong;

  4. Start Kong:

    $ kong start

# Kong is running
$ curl 127.0.0.1:8001

  5. Use Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.
