  • 12.04 Precise APT

  • 15.04 Vivid APT
  • 16.04 Xenial APT

Installation:

  1. Install the Package:

    If you are downloading the package, execute:

    $ sudo apt-get update
$ sudo apt-get install netcat openssl libpcre3 dnsmasq procps perl
$ sudo dpkg -i kong-0.9.2.*.deb

  2. Configure your database

    Configure Kong so it can connect to your database. Kong supports both PostgreSQL 9.4+ and Cassandra 2.2.x as its datastore.

    If you are using Postgres, please provision a database and a user before starting Kong, ie:

    CREATE USER kong; CREATE DATABASE kong OWNER kong;

  3. Start Kong:

    $ kong start

# Kong is running
$ curl 127.0.0.1:8001

  4. Use Kong:

    Quickly learn how to use Kong with the 5-minute Quickstart.
